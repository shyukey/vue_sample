### ZipLib検証

画面タブのAboutが検証画面です。
プロジェクト直下の検証.zip使ってください。
暗号化zip→アップ→url発行→urlから取得からダウンロード出来るファイルはzipなので要件には合うはず。
１．「SELECT IMAGE」でzipファイルを選択
２．「Q」か「A」か選択できるようにしてるので、選択で画像切り替え。
